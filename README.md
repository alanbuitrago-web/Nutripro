# NutriPro

Aplicación desarrollada en Python y Streamlit para apoyar la evaluación nutricional de deportistas principiantes.

## Funcionalidades

* Cálculo de indicadores nutricionales.
* Visualización de resultados mediante gráficos interactivos.
* Recomendaciones básicas de alimentación.
* Interfaz web desarrollada con Streamlit.

## Tecnologías utilizadas

* Python
* Streamlit
* Plotly
